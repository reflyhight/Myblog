###    数（整数，浮点，复数）
####   整数
如0，100，-100，整数有多重表示方式
如0，100，为默认的十进制表示方式，0x开头的16进制，0b开头的二进制
```
>>> int10=100
>>> int10
100
>>> int16=0xff00
>>> int16
65280
>>> int2=0b111111
>>> int2
63
```
####   浮点
浮点数，就是数学里的小数，因为表示的方式可以用科学计数法呈现不同的小数点位置，比如，1.23xe9和12.3xe8是完全相等的,e9代表10的9次方。在python中不区分单双精度。
```
>>> f1=3.3
>>> type(f1)
<type 'float'>
>>> f2=1.1e10
>>> type(f1)
<type 'float'>
>>> f1
3.3
>>> f2
11000000000.0
```
####   复数
在数学里，复数是数的一种表示方式，用a+bi方式表示，a,b是实数。而实数则是生活中存在的数，0，-1，1.1都是实数。虚数则是一种理论数字，如果一个数的平方是负数，那么这个数是虚数。可以用a+bi表示这个虚数。所有的虚数都只能用复数的方式表示。而实数也可以用复数的形式表示。如1+0i，表示的实数就是1，但它的数据类型却是复数。在python中用j代表数学中的i
```
>>> fu=1+1j
>>> fu
(1+1j)
>>> type(fu)
<type 'complex'>
>>> fu2=1+0j
>>> fu2
(1+0j)
>>> print(fu2==1)
True
```
 
 
 
###    布尔
True 和 False两种，非黑即白，就只有这两种。除此之外，其他类型可以隐身转换为布尔型。数型中0转换为False，除0之外转换为True,对象中None为False,其他为True。字符串中，空字符串也为False
 
```
>>> if(""):
...     print "true"
... else:
...     print "false"
...
false
 
>>> if("ok"):
...     print "true"
... else:
...     print "false"
...
true
 
>>> if(None):
...     print "true"
... else:
...     print "false"
...
false
```
 
###    字符串
字符串就是普通的字符序列。有几种常见的表示方法。注意，在python中并没有字符的概念，虽然字符串是由字符组成，但是并没有单独将字符列为一种数据类型，你要是非要强行理解字符的话，你可以认为类似JAVA中的字符在python里是由一个字符构成的字符串
####    单引号
如'Quote me on this'    ，在单引号中的字符串使用单引号需要转义，使用双引号不会让解释器产生歧义，是合法的
 
####   双引号
如"Quote me on this"，双引号并不是两个单引号组成，而是独立的一个键，所以双引号中有单引号也不会让解释器产生歧义，是合法的
 
####   三引号（三个单引号，或者三个双引号）
如'''I am haima'''   ，或者"""I am haima """。三引号定义的字符串可以跨行。而另外两种方式需要使用\n表示换行符，不允许跨行。
 
```
>>> """I am haima """
'I am haima '
>>> '''I am haima'''
'I am haima'
>>> '''haima
... oh'''
'haima\noh'
```
####   转义字符
通过\可以转义表示很多字符，\n换行，\t制表符，而\\表示字符串\，而如果文字中有特别多的单引号或者双引号，转义符\就会变得特别麻烦。通过在字符串前面加上R或者r的方式使得字符串编程自然字符串，字符串中的转义字符将不再有转义功能，当然也使得单引号表示的方法中无法出现单引号，双引号的字符串中无法出现双引号，因为无法转义
```
>>> r"Newlines are indicated by \n"
'Newlines are indicated by \\n'
```
 
####   字符编码
首先要明白`字符编码`的概念。在计算机最底层，数据只由于0和1表示，也就是我们所说的二进制。计算机最基本的表达就是数字。而将数字和字符一一映射起来，就成了一张字符编码表。如字母A用ASCII编码是十进制的65，用二进制表示为01000001。然而由于编码表并不是唯一的，不同的开发者按照各自的规则指定了不同的编码表。如支持中国字符的gbk，支持日本字符的Shift_JIS，等。常见字符编码有如下：
- ASCII ：只能表示255个字符，最早的计算机采用8个bit位作为1个字节，所以只能代表255个字符(8个bit位最大为11111111，255),包括数字0-9，字母a-z,A-Z,还有一些控制符。更多参考[百科](https://baike.baidu.com/item/ASCII/309296)
- gbk：最开始为支持中文编码而发明的
- Unicode：统一编码，万国码，把所有语言字符都囊括其中。仅仅是gbk并不能解决字符的问题，如果在一段文字中既要有汉字，又要有韩文，该怎么办呢。Unicode因为要囊括的字符比较多，所占字节为6个字节（廖雪峰官网所说，我不敢确定，暂时这样认为吧）。汉字在unicode中的编码范围为 4E00-9FA5(十六进制)，也就是19968-20901，算起来只能容纳两万多个汉字，我去查询了一下，汉字个数多大八九万个之多，这里我暂时不深究，先暂时理解到这里。
- utf-8：utf-8是Unicode的优化版本，可变长度编码，这样做的好处就是既能编写各国语言的字符，又能节约存储空间
 
 
####   字符转换
由于编码表的不同，产生乱码的问题很常见。要知道其中转换才能避免这个问题。unicode转化为其他编码叫encode,其他编码字符转换为unicode叫decode。
 
<img src="http://ou9e0q35h.bkt.clouddn.com/0.47091735899448395.png" style="width:300px;border:2px solid #C6C6C6;" >
如何将gbk的字符转换为utf-8编码字符呢。首先由gbk字符的字符decode为unicode，在由unicode编码字符encode为utf-8
 
###   变量
上面讲的都是独立的数据，如果我们想二次使用一个数据，便产生了变量，给一个数据取个名字，就形成的变量，如`a=1`就申明了一个变量a，在`Python编程（一）认识python`中的`标识符`小节中已经说名了变量名需要遵循标识符规范。python是弱数据类型的语言，在申明变量的时候，我们不需要像类似Java等语言需要给出变量的类型。解释器会自行判断数据类型
 
###    类型转换
不同的数据类型可以相互转换，如字符串的"123"可以转换成数字的123
```
a="123"
b=int(a)
```
类似的还有  str()，float()，bool(),complex()
```
a=1
b=complex(a)
b
(1+0j)
```
数据类型的转换，其实是调用的[内建函数（Built-in Functions）](https://docs.python.org/3/library/functions.html)
 
###    运算符

算数运算符

|  符号     |    名 |  描述 |
| :-------- | :--------:| :-- |
|   +   |   加   |   两个对象相加 a + b 输出结果 30
|  -  |    减   |   得到负数或是一个数减去另一个数 a - b 输出结果 -10
|  *   |   乘  |    两个数相乘或是返回一个被重复若干次的字符串 a * b 输出结果 200
|  /  |    除  |    x除以y b / a 输出结果 2
|  %  |    取余  |    返回除法的余数 b % a 输出结果 0
|  **  |    幂  |    返回x的y次幂 a**b 为10的20次方， 输出结果 100000000000000000000
|  //  |    取整除  |    返回商的整数部分 9//2 输出结果 4 , 9.0//2.0 输出结果 4.0

|  /=   |   除法赋值运算符 |     c /= a 等效于 c = c / a

|  \*\*=  |    幂赋值运算符  |    c \*\*= a 等效于 c = c \*\* a
